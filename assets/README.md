# Assets Folder

## Required Images:

Place these images in this folder before building for production:

1. **icon.png** (1024x1024)
   - App icon for both iOS and Android
   - Should be PNG with transparent background

2. **splash.png** (1284x2778)
   - Splash screen image
   - Portrait orientation

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Foreground layer only

4. **favicon.png** (48x48)
   - Web favicon
   - For web version

## For Testing:

The app will work without these images during development.
Expo will use default placeholders.

## To Generate Icons:

You can use online tools:
- https://appicon.co/
- https://www.appicon.build/
- https://makeappicon.com/

Or create manually in design software (Figma, Photoshop, etc.)
